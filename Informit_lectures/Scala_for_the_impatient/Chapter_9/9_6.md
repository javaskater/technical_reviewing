# Writing a file
* scala soes not it you have to use java classes
  * here we use the very simple _java.io.PrinWriter_ class
## Code
```scala
package com.absile.scala.chap09

import java.io.PrintWriter

object Test9_6 extends App {
  val out = new PrintWriter("/home/jmena01/ERICA/TOPAD/TOPAD-Cible/n.log", "UTF-8")
  for(i <- 1 to 100) out.println(i)
  out.println("à")
  out.println("\u7684")
  val price = 10.35
  val quantity = 5
  out.print(f"I paid $price%10.2f for $quantity%6d items\n")
  out.close()
}
```

## back to the f-formatting
* In the REPL:
```scala
scala> val price = 5.3546
val price: Double = 5.3546

scala> val quantity = 5
val quantity: Int = 5

scala> val s = f"$price%10.2f $quantity%6d"
val s: String = "      5,35      5"

scala> val s = f"prix: |$price%10.2f| pour |$quantity%6d|" //10 is the total fractional part included
val s: String = prix: |      5,35| pour |     5|
```
