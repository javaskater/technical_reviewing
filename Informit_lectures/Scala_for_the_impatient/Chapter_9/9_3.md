# reading doubles

* The example by using ma explains well what yield does
```scala
scala> val source = Source.fromFile("/home/jmena01/ERICA/TOPAD/TOPAD-Cible/doubles.txt", "UTF-8")
val source: scala.io.BufferedSource = <iterator>

scala> val tokens = source.mkString.split("\\s+")
val tokens: Array[String] = Array(12.4, 34.56, 45.6, 78.9)

scala> val numbers = for(w <- tokens) yield w.toDouble
val numbers: Array[Double] = Array(12.4, 34.56, 45.6, 78.9)
scala> tokens //token has not been emptied by the read !!!
val res0: Array[String] = Array(12.4, 34.56, 45.6, 78.9)

scala> val other_numbers = tokens.map(_.toDouble)
val other_numbers: Array[Double] = Array(12.4, 34.56, 45.6, 78.9)
```
