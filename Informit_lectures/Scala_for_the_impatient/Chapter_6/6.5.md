## Adding the  duration of the program execution
* contrary to what is written _-Dscala.time_ is not a scala option but more a JVM Option
* On the IntelliJ Run configuration Window (_Modify run configuration_):
  * the drop down list _Modify options_
  * at the end of the Java's Part of the dropdown list
    * check _Add VM Option_
  * In the new Input field enter _-Dscala.time_
