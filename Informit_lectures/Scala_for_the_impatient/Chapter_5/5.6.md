# p 64

```scala
scala> class Person(val name:String, val age: Int){
     | println("Just constructed another Person");
     | def description() = s"$name is $age years old";
     | }
defined class Person

scala> val p1 = new Person("Pinzoni", 80)
Just constructed another Person
p1: Person = Person@12a148d0

scala> p1.description
res14: String = Pinzoni is 80 years old

scala> p1.description()
res15: String = Pinzoni is 80 years old
```
* if inside the Person class we define _description_ without __()__
  * we only can call _p1.description_ not _p1.description()_
# p 65
* [official Scala Documentation about BeanProperty](https://www.scala-lang.org/api/2.13.3/scala/beans/BeanProperty.html)
```scala
scala> import scala.beans.BeanProperty;
import scala.beans.BeanProperty

scala> class Person(@BeanProperty var name:String ="ZORRO", @BeanProperty var age: Int = 26){
     | println("Just constructed another Person");
     | def description = s"$name is $age years old";
     | }
```
* With _@BeanProperty_ the fields are by default private !!!!
* _In Scala, classes take parameters, just like methods do._
