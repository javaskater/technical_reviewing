* Any is not Unit
```scala
scala> def printAny(x:Any):Unit = {println(x)}
def printAny(x: Any): Unit

scala> def printUnit(x:Unit):Unit = {println(x)}
def printUnit(x: Unit): Unit

scala> printAny("Hello")
Hello

scala> printUnit("Hello")
                 ^
       warning: a pure expression does nothing in statement position

```
* When many parameters are given and an Object is expected the are put in a tuple object:
```scala
cala> def show(o:Any):Unit = {println(s"${o.getClass}: $o")}
def show(o: Any): Unit

scala> show(3)
class java.lang.Integer: 3

scala> show(3, 4, 5, 6)
class scala.Tuple4: (3,4,5,6)
```
