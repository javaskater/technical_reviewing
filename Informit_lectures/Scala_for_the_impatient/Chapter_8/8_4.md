* a primary constructor is not a default constructor
  * it is not a constructor with no parameter
  * a primary constructor an call any superclass constructor (not only the superclass primary constructor) 
# 95 up
* The primary constructor defines private attributes (see equivalent Java Class)
  * I tested on the Scala REPL
  * by default all constructor attributes (without val or var) make private val members

* I can mutate a var parameter
  * because (see chapter 5) its aggessor methods are automatically created:
```scala
scala> class Person(var age:Int){
     | def show():Unit
     | = {
     | println("age is:"+age)
     | }
     | }
class Person

scala> val p = new Person(24)
val p: Person = Person@4cfc0c48

scala> p.show()
age is:24

scala> p.age = 45
// mutated p.age

scala> p.show()
age is:45
```
