# custom font:

* it is the __[2.18](https://github.com/CSSInDepth/css-in-depth-2/blob/2nd-edition/ch02/listing-2.18.html)__ script

```css
/** Define it in a scope*/
:root {
    --main-font: Helvetica, Arial, sans-serif;
}
/**Use it */
p {
    font-family: var(--main-font);
}
```
* We can have default value when the property is not defined:
  * if the --brand-color variable is not defined we fallback to blue (_fallback value_)
  * tested by commenting out the _--brand-color variable_ definition
```css
p{
    font-family: var(--main-font);
    color: var(--brand-color, blue);
}
```
