# p 230
* To see the style changing I change the color of the *h1* title in the media query
```css
.title > h1 { /* still in the header */
    color: #333;
    text-transform: uppercase;
    font-size: 1.5rem;
    margin-block: 0.2em;
    /*text-shadow: 0.5em 0.5em 0.2em green;*/
}
@media (min-width: 560px){
    .title > h1 {
        font-size: 2.25rem;
        color: #b81414;
    }
}
```
* The media query is placed after the tyle to override
  * because the priority is *based on selector specificity or source order*

# p 235
* the *!important* keyword is meant to avoid worrying about the cascade overriding the color or the background
```css
@media print {
    * {
        color: black !important;
        background: none !important;
    }
}
```
