# Image

* _@_ introduces a context relative path
* To run the webSite
  * The Goal is __spring-boot:run__

```bash
$ ./mvnw spring-boot:run
```
* whe then go to _http://localhost:8080/_

# Testing
* Like explained in this [Eclipse Blog article](https://www.vogella.com/tutorials/JUnitEclipse/article.html)
	* The test launching can be done via _ALT+Shift+X_
*  To see the JUnit Tab while launching test:
  * _Window / Show view_
  * _Others / Java / JUnit_
  
# The Spring Boot Dashboard
* if the _Boot Dashboard_ tab is not present
* You can add it by 
  * _Window / Show view_
  * _Others / Other / Boot Dashboard_
